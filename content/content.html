Penerapan Flexbox

Apa itu flexbox?

Terdapat banyak cara untuk mengatur sebuah layout pada halaman html, salah satunya adalah FlexBox. Metode flexbox menggunakan property display bernama flex.

Flex sangat cocok digunakan untuk komponen sebuah layout skala kecil. Tujuan utama dari display ini adalah memberi kemampuan container agar dapat memanipulasi tinggi, lebar, urutan posisi element, dan juga jarak (Space).

Untuk mengatur sebuah layout diatas kita memerlukan sebuah property display flex pada container atau pembungkus dari element di dalamnya.
